PORTFOLIO


S.C.A.N. (Supporters of Children With Special Needs) APPLICATION

S.C.A.N. is a full-stack application that employs CRUD functionality, Thymeleaf, Spring Boot, HTML, Javascript, CSS, Bootstrap, Java, mySQL and Integration Tests to create a modernized and secure website for their client S.C.A.N. (Supporters of Children with Additional Needs). The revamped website aims to be mobile-responsive and user-friendly, allowing the non-profit to receive donations, post events that users can volunteer for, and allow the them to share photos of events.

DOGLISTER APPLICATION

Full-stack dog adopting application. Users create dog listings, post or search dog listings. Users can also edit and delete their own dog listings. Programmed utilizing MVC design pattern, Java Servlets, JSP, JSTL expression language, password hashing, and mySQL database.



WEATHER MAP APPLICATION

A weather application where users can get a five day forecast for any city in the U.S. They can also move a location marker on a map to a place they want to find a five day forecast for. Utilized jQuery, HTML, MapBox API, Bootstrap, and Open Weather Map API.
